Política de Privacidad

Última actualización: 17 de julio de 2025

Nombre de la aplicación: Solar Group

Desarrollador: S&F Solar Group SAC

1. Introducción

Esta aplicación está destinada exclusivamente para uso interno por parte del personal autorizado de S&F Solar Group SAC. La protección de los datos personales es una prioridad para nosotros. Esta política explica qué datos recopilamos, cómo los usamos y cómo los protegemos.

2. Datos que recopilamos

  • Información de usuario: nombre, correo electrónico corporativo (solo para autenticación).
  • Datos de ventas u oportunidades comerciales: contactos, ubicación, notas.
  • Ubicación (opcional): si el usuario lo permite, para asociar oportunidades de venta a una ubicación geográfica.

3. Uso de la información

Los datos se utilizan para registrar oportunidades comerciales internas, identificar usuarios autenticados y mejorar el seguimiento de prospectos.

4. Compartición de datos

No compartimos datos con terceros. Toda la información permanece dentro de la infraestructura de S&F Solar Group SAC.

5. Seguridad

Protegemos los datos mediante autenticación segura y medidas de seguridad razonables.

6. Retención de datos

Los datos se retienen solo mientras sean necesarios para los fines mencionados o por requisitos internos.

7. Derechos del usuario

Los usuarios pueden solicitar acceso, corrección o eliminación de sus datos contactando al equipo de soporte.
